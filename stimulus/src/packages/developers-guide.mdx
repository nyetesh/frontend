# Developers' Guide

## Style Guides

1. For stylesheet
- Start a class name with `scl-` prefix.
```scss
.scl-button {
    border: 0;
}
```

2. For angular component
- Start a `selector` with `scl-` prefix.
- Start a name component, directive, pipe and module with `Scl` prefix
```ts
@Component({
    selector: 'scl-button',
    templateUrl: 'button.html',
    standalone: true,
    host: {
        class: 'scl-button'
    }
})
export class SclButton { }
```
