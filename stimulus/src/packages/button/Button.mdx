# Button

Use nebular directive for button.  
*Ref: [https://akveo.github.io/nebular/docs/components/button/overview#nbbuttoncomponent](https://akveo.github.io/nebular/docs/components/button/overview#nbbuttoncomponent)*

```html
<button type="button" nbButton>Button</button>
```

## Fill Buttons
```html
<button type="button" nbButton [status]="primary">Button</button>
<button type="button" nbButton [status]="danger">Button</button>
<button type="button" nbButton [status]="basic">Button</button>
```
